圆角

定义元素的圆角半径radius

元素 属性 值 描述
任何元素 class radius4 | radius6 | radius8 | radius16 | radius20 | radius40 | radiusround 圆角半径

高度

定义了元素的高度height

元素 属性 值 描述
任何元素 class height26 | height30 | height32 | height40 | height50 | height54 定义宽高，字体大小

元素尺寸

定义了元素的宽高尺寸size

元素 属性 值 描述
i标签 class size8 | size12 | size16 | size20 | size24 | size28 | size30 | size32 | size40 | size54 定义宽高，字体大小
头像控件 class photo | size32 | size40 | size50 | size54 | size80 照片 | 定义宽高，和photo内部字体大小
徽章控件 class badge | size32 徽章 | 定义总宽度
p标签(文字) class size17 | size20 文字大小

间距

定义了元素的高度margin|padding|space|rowspace|indent

元素 属性 值 描述
任何元素 class margin4 | margin6 | margin8 | margin12 | lrmargin8 | nomargin | nolrmargin | notbmargin 定义外间距
任何元素 class padding4 | padding6 | padding8 | padding12 | lrpadding8 | nopadding | nolrpadding | notbpadding 定义内间距
任何元素 class space4 | space6 | space8 | space12 | space14 | space16 宽度间距
任何元素 class rowspace4 | rowspace6 | rowspace8 | rowspace12 | rowspace20 | rowspace40 高度间距
任何元素 class indent6 | indent8 | indent24 道行缩进
任何元素 class lineheight0 行高为0

文字

定义了文字的样式title|sliver|article-title|small|paragraph|capitalize|uppercase|lowercase|bold

元素 属性 值 描述
任何元素 class title 标题
任何元素 class sliver 条子标题
任何元素 class article-title 文章标题
small class 无 副标题
任何元素 class paragraph 段落
任何元素 class capitalize 首字母大写
任何元素 class uppercase 大写
任何元素 class lowercase 小写
任何元素 class bold 加粗

对齐

定义了很多元素对齐方式box..|self-middle..|horizontal|float-left..|tb..|text-left..

元素 属性 值 描述
div弹性盒子(盒子内部) class box | box-left | box-center | box-right | box-top | box-middle | box-bottom | box-middlecenter 弹性盒子 | 自适应区域 | 左对齐 | 水平居中 | 右对齐 | 上对齐 | 垂直居中 | 下对齐 | 水平垂直居中
弹性盒子内部元素 class box-flex-1 | box-flex-2 | box-flex-3 | justify | justify-content 弹性率1 | 弹性率2 | 弹性率3 | 与justify-content搭配最大占用90px的宽度 | 与justify搭配，自适应剩余宽度
任何元素(自身) class self-middle | self-center | self-middlecenter 垂直居中 | 水平居中 | 垂直水平居中
任何元素(内部文字) class tb | bt | tb-rl | tb-lr | lr | rl 上下 | 下上 | 上下，左右 | 上下，右左 | 左右 | 右左
div标签(内部元素) class horizontal 水平排列
任何元素(自身浮动) class float-left | float-right 左浮动 | 右浮动
内部文字 class text-left | text-center | text-right 左对齐 | 水平居中 | 右对齐

边框

定义了控件线条的样式border|dotted|underline

元素 属性 值 描述
hr class 无 | dotted 显示实线 | 显示虚线
ul class dotted 子级li底部显示虚线
li class dotted 底部显示虚线
.list控件 class list | border-content | dotted 列表控件 | 内容部分显示实线 | 底部虚线
.photo控件 class photo | border 头像控件 | 边框显示实线
.timeline控件 class timeline | border | dotted 时间轴控件 | 每行底部显示实线 | 每行底部显示虚线
.card>.card-handler控件 class card-handler | border 卡片操作区 | 操作按钮间显示实线分割
.title控件 class title | border 标题 | 底部显示实线
.grid控件 class grid | border 栅格控件 | 栅格间显示实线
.input-box控件 class underline 下划线
任何元素 class noborder 去线或者去边框

点击状态

定义了元素点击状态click|noclick

元素 属性 值 描述
SeedsUI中所有含有点击效果的元素 class noclick 无点击
.input-box class click 添加点击效果

每行列数

定义了元素每行列数

元素 属性 值 描述
.card>.card-handler控件 data-col 2,3,4 每行列数
包含.tab的元素 data-col 2,3,4,5,6 每行列数
.grid栅格控件 data-col 2,3,4,5 每行列数
