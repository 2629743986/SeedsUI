导航

用于头部导航、搜索导航、页签导航和底部导航。

头部导航

<div class="titlebar">
    <a href="javascript:history.go(-1)">
        <i class="icon icon-arrowleft"></i>
    </a>
    <h1>左右图标titlebar</h1>
    <a>
        <i class="icon icon-rdoadd"></i>
    </a>
    <a>
        <i class="icon icon-search"></i>
    </a>
</div>

头部导航titlebar以自适应方式布局，在此容器内的元素都将 排成一行 ，而文字区域为自适应的区域， 其它区域越大，文字区域就越小 ，具体示例： bar.html 页面。|

元素 属性 值 描述
I.导航盒子div class .titlebar 导航盒子
.reverse 盒子的颜色反转
II.文字区域h1 class 无 自动被左侧元素挤压
.text-center 文字水平居中
II.左侧元素a或button class 无 绝对定位于左侧，不占用区域
.left 占用左侧区域
II.右侧元素a或button class 无 占用右侧区域
.right 绝对定位于右侧，不占用区域

单页模式

单页模式包含:section容器间动画跳转、返回的功能。它会监听来判断android返回按键，并提供相关回调

HTML

<section id="page_single_0" class="active">
  <a href="#page_single_1" data-target="section">进入第二页</a>
</section>
<section id="page_single_1" data-sectionanimation="slideleft">
  <a href="#page_single_1" data-target="section">进入第三页</a>
</section>
<section id="page_single_2" data-sectionanimation="slideright">
  第三页
</section>
元素 属性 值 描述
section标签 data-sectionanimation none 无动画
slideleft 左右
slideright 右左
slideup 上下
slidedown 下上
fade 淡入淡出
data-target section 用单页模式打开
任何元素 href #id 连接等于section的ID
data-target section 用单页模式打开

Javascript

var sec=new Section({
  "onSectionOpenStart":function(e){
    console.log("您点击了：");
    console.log(e.target);
    console.log("页面加载前"+e.sectionId);
  },
  "onSectionOpenEnd":function(e){
    console.log("您点击了：");
    console.log(e.target);
    console.log("页面加载完成"+e.sectionId);
    console.log(e.history);
  },
  "onSectionCloseStart":function(e){
    console.log("您点击了：");
    console.log(e.target);
    console.log("页面关闭前"+e.sectionId);
  },
  "onSectionCloseEnd":function(e){
    console.log("您点击了：");
    console.log(e.target);
    console.log("页面关闭完成"+e.sectionId);
  }
});

Section初始化，支持下列参数

参数 类型 默认值 描述
onSectionOpenEnd Callback 无 页面加载完成
onSectionCloseStart Callback 无 页面关闭前
onSectionCloseEnd Callback 无 页面加载前
onSectionOpenStart Callback 无 页面关闭完成
方法 参数 描述
openSection String sectionId 根据id打开传入的section
backSection 无 返回上一页
closeSection String sectionId,String animation(可不填) 关闭指定的section，可以设置动画

var actionsheetParams=[
  {
    text : '菜单一',

          console.log(s.target);
          console.log("您点击了菜单一");
        }
  },{
    text : '菜单二',
    handler : function(s){
          console.log(s.target);
          console.log("您点击了菜单二");
    }
    },{
    text : '取消',
    handler : function(s){
          console.log(s.target);
          console.log("您点击了取消");
          s.hide();
    }
];
var actionsheet=new Actionsheet();
$("#actionsheet").click(function(){
  actionsheet.setParams(actionsheetParams);

//数字计数器
function listenerFontcount(){
  var fc=new FontCounter({
    "onInput":function(s){
      //console.log("输入中");
    },
    "onMoreThanCount":function(s){
      //console.log("超过数字限制时");
      s.fieldCounter.innerHTML=Math.round(s.maxlength-s.currentlength);
      s.fieldCounter.classList.add("color-tip");
    },
    "onNotMoreThanCout":function(s){
      //console.log("没有超过数字限制时");
      s.fieldCounter.innerHTML="";
      s.fieldCounter.classList.remove("color-tip");
    }
  });
}

滑动容器

滑动容器只支持触屏滑动，可支持循环滑动、一屏多页等滑动模式，详见slider.html页面和slider-page.html页面

html

<div class="slider-container" id="carousel1">
    <div class="slider-wrapper">
        <div class="slider-slide">
            <img src="../img/components/banner.jpg"/>
            <div class="slide-title">新闻标题1</div>
        </div>
        <div class="slider-slide">
            <img src="../img/components/banner.jpg"/>
            <div class="slide-title">新闻标题2</div>
        </div>
    </div>
    <div class="slider-pagination"></div>
</div>

Javascript

var s1=new Slider("#carousel1",{
    "pagination":".slider-pagination",
    "loop":true
});

new Slider(container,params)初始化，支持下列两个参数container和params

参数1:String container

参数 类型 默认值 描述
container String 无 滑动容器的id

参数2:Json params

参数 类型 默认值 描述
pagination String null .class|#id，滑动容器内部的滑动页
autoplay Number 无 毫秒数，设置自动播放的毫秒数
slidesPerView Number 1 一屏几页，默认一屏1页
threshold Number 50 滑动多少像素，则翻页
duration Number 300 翻页动画时长
loop Boolean false true|false，设为true则循环
slideDuplicateClass String "slider-slide-duplicate" 循环状态时，复制出来的slide的额外增加的class名称，用于标识出复制的slide
wrapperClass String "slider-wrapper" 画布class名称
slideActiveClass String "active" 激活状态slide的class名称
bulletClass String "bullet" 小点点的class名称
bulletActiveClass String "active" 激活状态小点点的class名称
onInit Callback 无 滑动容器初始化完成时回调
onSlideChangeStart Callback 无 开始滑动时回调
onSlideChangeEnd Callback 无 滑动结束后回调

下拉刷新

下拉刷新除了提供下拉刷新外，还提供了滚动至底部刷新

new Dragrefresh(container,params)初始化，支持下列两个参数container和params

参数1:String container

参数 类型 默认值 描述
container String 无 下拉刷新容器的id、class、标签、DOM皆可

参数2:Json params

参数 类型 默认值 描述
refreshThreshold Number 100 下拉多少像素刷新
refreshThresholdMax Number 200 最大能拉多少像素
refreshHideTop Number 0 下拉完成后，收缩到什么位置
duration Number 300 下拉完，收缩动画时长
timeout Number 5000 超时毫秒
onRefreshStart Callback 无 开始刷新时回调
onRefreshEnd Callback 无 刷新完成时回调
onRefreshTimeout Callback 无 刷新超时回调
onBottom Callback 无 滚动到底部时回调

仪表盘

提供了可视化数据仪表盘

Html

<div class="gauge-box" id="gauge">
    <div class="gauge">
        <div class="gauge-pointer"></div>
        <div class="gauge-text">0</div>
    </div>
</div>

Javascript

var g=new Gauge("#gauge",{
    minValue:100,
    maxValue:400,
    currentValue:200
});

new Gauge(container,params)初始化，支持下列两个参数container和params

参数1:String

参数 类型 默认值 描述
container String|DOM 无 仪表盘容器的参数用id、class、标签皆可
onTap Function 无 点击树结构回调
onTapLastChild Function 无 点击树结构底层节点回调

参数2:Json params

参数 类型 默认值 描述
minValue Number 0 最小值
maxValue Number 360 最大值
currentValue Number 0 当前值
pointClass String ".gauge-pointer" 指针class名称
waveClass String ".gauge-wave" 波浪class名称
valueClass String ".gauge-value" 值class名称
durationall Number 2000 从最小值到最大值动画时长，一般动画时长低于此值
onInit Callback 无 完成初始化时触发
onPointChangeStart Callback 无 指针开始旋转时触发
onPointChangeEnd Callback 无 指针完成旋转时触发
