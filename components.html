导航

用于头部导航、搜索导航、页签导航和底部导航。

头部导航

<div class="titlebar">
    <a href="javascript:history.go(-1)">
        <i class="icon icon-arrowleft"></i>
    </a>
    <h1>左右图标titlebar</h1>
    <a>
        <i class="icon icon-rdoadd"></i>
    </a>
    <a>
        <i class="icon icon-search"></i>
    </a>
</div>

头部导航titlebar以自适应方式布局，在此容器内的元素都将 排成一行 ，而文字区域为自适应的区域， 其它区域越大，文字区域就越小 ，具体示例： bar.html 页面。

元素 属性 值 描述
I.导航盒子div class .titlebar | .reverse 导航盒子 | 盒子的颜色反转
II.文字区域h1 class 无 | .text-center 自动被左侧元素挤压 | 文字水平居中
II.左侧元素a或button class 无 |.left 占用左侧区域 | 绝对定位于左侧，不占用区域
II.右侧元素a或button class 无 |.right 占用右侧区域 | 绝对定位于右侧，不占用区域

搜索导航

<div class="searchbar">
    <div class="input-box radius20" data-input="clear">
        <input type="search" placeholder="搜索框" class="search">
        <i class="icon icon-clear-fill color-placeholder"></i>
    </div>
    <a>搜索</a>
</div>

搜索导航searchbar以自适应方式布局，在此容器内的元素都将 排成一行 ，而文字区域为自适应的区域， 其它区域越大，文字区域就越小 ，具体示例： bar.html 页面。

元素 属性 值 描述
I.搜索盒子div class .searchbar 搜索盒子

单页模式

单页模式包含:section容器间动画跳转、返回的功能。它会监听来判断android返回按键，并提供相关回调

HTML

<section id="page_single_0" class="active">
  <a href="#page_single_1" data-target="section">进入第二页</a>
</section>
<section id="page_single_1" data-sectionanimation="slideleft">
  <a href="#page_single_1" data-target="section">进入第三页</a>
</section>
<section id="page_single_2" data-sectionanimation="slideright">
  第三页
</section>
元素 属性 值 描述
section标签 data-sectionanimation none | slideleft | slideright | slideup | slidedown | fade 无动画 | 左右 | 右左 | 上下 | 下上 | 淡入淡出
data-target section 用单页模式打开
任何元素 href #id 连接等于section的ID
data-target section 用单页模式打开

Javascript

var sec=new Section({
  "onSectionOpenStart":function(e){
    console.log("您点击了：");
    console.log(e.target);
    console.log("页面加载前"+e.sectionId);
  },
  "onSectionOpenEnd":function(e){
    console.log("您点击了：");
    console.log(e.target);
    console.log("页面加载完成"+e.sectionId);
    console.log(e.history);
  },
  "onSectionCloseStart":function(e){
    console.log("您点击了：");
    console.log(e.target);
    console.log("页面关闭前"+e.sectionId);
  },
  "onSectionCloseEnd":function(e){
    console.log("您点击了：");
    console.log(e.target);
    console.log("页面关闭完成"+e.sectionId);
  }
});

Section初始化，支持下列参数

参数 类型 默认值 描述
onSectionOpenEnd Function 无 页面加载完成
onSectionCloseStart Function 无 页面关闭前
onSectionCloseEnd Function 无 页面加载前
onSectionOpenStart Function 无 页面关闭完成
方法 参数 描述
openSection String sectionId 根据id打开传入的section
backSection 无 返回上一页
closeSection String sectionId,String animation(可不填) 关闭指定的section，可以设置动画

var actionsheet=new Actionsheet();

小弹框

var popToast=new Toast("小弹框");
$("#toast").click(function(){
  popToast.show();
});

Toast初始化，支持下列两个参数msg和params

参数1:String msg

参数 类型 默认值 描述
msg String 无 提示文字

参数2:Json params

参数 类型 默认值 描述
delay Int 1000 显示时长
方法 参数 描述
setText String msg 修改提示文字
hide 无 隐藏
show 无 显示
destory 无 清除

自定义弹出框

$("#pop_custom").dialog({
    onClick:function(s){
        if(s.target.text=="取消"){
            s.hideDialog();
        }else if(s.target.text=="确定"){
            console.log("确定");
        }
    }
});

dialog扩展$，支持下列params

参数:Json params

参数 类型 默认值 描述
onClick(Dialog) Function 无 点击回调
方法 参数 描述
showDialog 无 显示
hideDialog 无 隐藏
